# Jurisdiction: GDPR (EU / EEA)

**Statute:** General Data Protection Regulation (EU) 2016/679
**Right:** Right of access — **Article 15**; portability — **Article 20**
**Regulator:** Competent national Data Protection Authority (DPA); coordinated via the European Data Protection Board (EDPB).

## Response clock

- **1 month** from receipt of the request (Art. 12(3)).
- Extension: **up to 2 further months** where necessary given complexity/number of requests; the controller must **inform the data subject within the first month** of the extension and the reasons.
- Source: [gdpr-info.eu — Right of access](https://gdpr-info.eu/issues/right-of-access/), [EDPB — Respect individuals' rights](https://www.edpb.europa.eu/sme-data-protection-guide/respect-individuals-rights_en)

## Cost

- **First copy free of charge** (Art. 15(3)). Reasonable fee only for further copies, or where requests are manifestly unfounded/excessive.

## Required elements

- Data subject request for a copy of all personal data processed + supplementary info under Art. 15(1) (purposes, categories, recipients, retention, source, existence of automated decision-making).
- Portability (Art. 20): structured, commonly used, machine-readable format where processing is by consent/contract and automated.

## Escalation path (maps to DSAR ladder)

- **R2 non-compliance:** 1-month (or extended 3-month with notice) deadline passes without adequate response.
- **R3 regulator:** lodge complaint with the data subject's national **DPA** (Art. 77). Cross-border cases handled via lead-DPA / one-stop-shop.

## Clock formula

`deadline = sent_date + 1 month` · `extended_deadline = sent_date + 3 months (only if extension notice received within 1 month)`
